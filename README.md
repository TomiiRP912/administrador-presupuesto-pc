# 🎮 Administrador de Presupuesto para PC Gamer

##  Descripción

Este proyecto consiste en una aplicación web desarrollada con **HTML, CSS y JavaScript** que permite planificar el armado de una PC Gamer controlando el presupuesto disponible.

El sistema permite agregar componentes, registrar su precio, clasificarlos por categoría y visualizar el gasto total en relación al presupuesto definido por el usuario.

El objetivo es ayudar a organizar la compra de los componentes de una computadora evitando superar el presupuesto establecido.

---

## Funcionalidades

La aplicación incluye las siguientes funciones:

* Agregar componentes de hardware (nombre, precio y categoría)
* Marcar si el componente ya fue comprado
* Visualizar los componentes agregados en una tabla
* Eliminar componentes de la lista
* Calcular automáticamente el total gastado
* Definir un presupuesto máximo
* Mostrar el presupuesto restante disponible
* Cambio de color cuando se supera el presupuesto

## Tecnologías utilizadas

Este proyecto fue desarrollado utilizando tecnologías web básicas:

* **HTML5** → estructura de la página

## Cómo utilizar la aplicación

1. Abrir el archivo **index.html** en un navegador web.
2. Definir un presupuesto máximo.
3. Agregar componentes indicando:

   * nombre
   * precio
   * categoría
   * estado de compra
4. Los componentes se mostrarán en la tabla.
5. El sistema calculará automáticamente el total gastado y el presupuesto restante.
